Punchcard Love

A five-man hack for HackNY. It's an Arduino LED light-sensor that reads custom punchcards that send texts, fetches the New York Times, and tweets. It's the new hipster way to use your computer.

You can check out the HackerLeague project, or browse the source code for the Arduino LED sensor on GitHub.

Celebrity Sticky Notes

A gang-of-five hack for Hack Princeton. Uses the OpenTok API, facial-recognition, and CSS black-magic transforms to render a sticky note on your forehead just so you can guess what's written on each other's heads.

It was never fully finished in 24 hours, but you can peruse the HackerLeague page and the source code.
